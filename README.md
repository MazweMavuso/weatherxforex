# 🌤️ SkyCast: Weather & Forex Platform

**[Live Demo](https://weatherxforex.vercel.app/)**

A professional-grade dashboard for weather and financial insights, **100% vibe-coded** using the **Gemini CLI model** directly from the command prompt.

## 🚀 The AI Software Engineer Mission
This project was created with a specific goal: **to improve and master AI model prompting skills.** 

The entire application—from the complex Glassmorphism UI to the dynamic Chart.js integration and real-time API handling—was built without writing a single line of manual code. It serves as a testament to the power of high-level strategic orchestration between a human and an AI agent.

## ✨ Key Features

### 🌦️ Weather Dashboard
- **Geolocation & Global Search:** Automatically detects your location or allows searching for any city worldwide.
- **Smart Suggestions:** Real-time city autocomplete to prevent misspellings.
- **AI-Generated Insights:** Provides friendly "Daily Reviews" (e.g., "Perfect day for a walk") based on weather data.
- **Advanced Metrics:** Displays Humidity, Wind Speed, and "Feels Like" temperatures.

### 💹 Forex Dashboard
- **Live Exchange Rates:** Real-time data for global currencies (USD, EUR, GBP, ZAR, JPY, etc.).
- **Dynamic Trend Analysis:** Interactive growth charts powered by Chart.js.
- **Custom Analysis:** Select any base/target currency pair to analyze trends over the current month.
- **Quick Converter:** A sleek, interactive currency converter with a "Swap" feature.

### 🎨 Design Language
- **Glassmorphism:** A clean, modern UI featuring translucent cards and background blurs.
- **Animated Backgrounds:** Sophisticated mesh-gradient "blobs" for a high-end feel.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
- **Professional Icons:** Integrated Lucide icon library for crisp, dynamic symbols.

## 🛠️ Built With
- **Vibe Coded via Gemini CLI**
- **HTML5 & Vanilla CSS3**
- **JavaScript (ES6+)**
- **APIs:** Open-Meteo (Weather), ExchangeRate-API & Frankfurter (Forex)
- **Library:** Chart.js, Lucide Icons
